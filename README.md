## Uber Eats Review Web Scraping

Web scraping project extracting customer review data from Trustpilot.

### What this project does
- Scrapes review dates and star ratings from Trustpilot's Uber Eats page
- Automates scraping across 5 pages using a for-loop
- Builds a clean data frame from the scraped data

### Tools used
R, rvest, tidyverse, xml2, purrr

### Key skills demonstrated
- HTML parsing with read_html()
- Extracting nodes and attributes from web pages
- String cleaning and data type conversion
- Loop automation with paste0 for multiple pages

### Data source
https://www.trustpilot.com/review/ubereats.com
